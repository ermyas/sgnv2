# Events

The farming module emits the following events:

## Handlers

### MsgClaimRewards

| Type    | Attribute Key | Attribute Value           |
|---------|---------------|---------------------------|
| claim   | address       | {userAddress}             |
| claim   | pool          | {poolName}                |

### MsgClaimAllRewards

| Type        | Attribute Key | Attribute Value           |
|-------------|---------------|---------------------------|
| claim_all   | address       | {userAddress}             |
